# Healthcare Monitoring AI Agent (React + FastAPI)

A modern, fast, and secure digital health assistant built with React, FastAPI, and LangChain.

## 🚀 Getting Started

### 1. Backend Setup
1. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Set up your `.env` file with `GROQ_API_KEY`.
3. Run the server:
   ```bash
   python backend/app.py
   ```

### 2. Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack
- **Frontend:** React, TypeScript, Vite, Lucide Icons.
- **Backend:** FastAPI, Python.
- **AI Agent:** LangChain, Groq (Llama 3.3).
- **Vector DB:** ChromaDB.
